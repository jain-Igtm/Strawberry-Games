# Model assets

Put optimized `.glb` models here.

Target mobile rules:

- Prefer `.glb` over `.gltf`.
- Keep individual tree models under ~5k triangles if possible.
- Use low-poly LOD variants for distant models.
- Use compressed textures when possible.
- Good starter folders:
  - `trees/`
  - `rocks/`
  - `buildings/`
  - `vehicles/`
  - `creatures/`

The game code is being prepared so these can replace procedural placeholder meshes.
