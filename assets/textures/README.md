# Texture assets

Put optimized textures here.

Target mobile rules:

- Use small textures first: 512x512 or 1024x1024.
- Ground textures should tile cleanly.
- Main categories:
  - `ground/grass/`
  - `ground/dirt/`
  - `ground/rock/`
  - `trees/bark/`
  - `trees/leaves/`
  - `sky/`

The next realism jump comes from replacing flat materials with real bark, leaf, rock, grass, and dirt textures.
