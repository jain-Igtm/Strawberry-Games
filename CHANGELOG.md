# Strawberry Forest Changelog

## Update 18

### Blocks and terrain
- Reworked distant ridge generation so mountains are part of the terrain instead of standalone pyramid landmarks.
- Added procedural terrain textures for grassland and stone ridges.
- Improved color blending on slopes, paths, wet ground, and high terrain.

### World generation
- Kept chunked world loading.
- Reduced obvious placeholder landmarks.
- Adjusted forest placement so trees avoid steep ridge surfaces.
- Water now appears only where the generated terrain calls for it.

### Interface
- Cleaned the loading text.
- Removed development reminder copy from the visible page.
- Updated the old build label into a coherent 1-N update number.

### Versioning
- This is the final 1-N update-numbered build.
- Future public builds will use semantic versioning.
- The next version will be v0.0.1.
